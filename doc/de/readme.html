QuietConsole NVDA-Add-on

QuietConsole hält NVDA in klassischen Konsolen (cmd.exe, conhost.exe) reaktionsschnell. Der globale Ruhemodus unterdrückt Caret-/Wert-/Text-/Namensänderungen und bricht laufende Sprachausgabe ab; ausgeschaltet verhält sich die Konsole wie gewohnt.

Tastenkombinationen

  • NVDA+Umschalt+C: Ruhemodus umschalten (global). Einschalten bricht Sprachausgabe ab.
  • NVDA+Umschalt+R: Letzte 30 Zeilen (konfigurierbar) lesen und die Review an das Ende setzen.

Einstellungen

NVDA → Einstellungen → Quiet Console:

  • Konsolen im Ruhemodus starten.
  • Zeilenanzahl für den "Letzte Zeilen lesen"-Befehl.
  • Unterdrückte Ereignisse protokollieren (Debug).

Alle Einstellungen wirken sofort und bleiben gespeichert.

Hinweise

  • Ruhemodus gilt global für alle Konsolen.
  • An = Ereignisse unterdrückt und Sprache abgebrochen; Aus = keine Unterdrückung.
