QuietConsole NVDA アドオン

QuietConsole はクラシックなコンソール（cmd.exe / conhost.exe）で NVDA を軽快に保つためのグローバル静音モードを提供します。オンではキャレット／値／テキスト／名前変更イベントを抑制し、進行中の読み上げをキャンセルします。オフでは通常の挙動です。

ジェスチャー

  • NVDA+Shift+C: 静音モードを切り替え（グローバル）。オンにすると読み上げをキャンセルします。
  • NVDA+Shift+R: 末尾30行（変更可）を読み上げ、レビューを末尾へ移動。

設定

NVDA → 設定 → Quiet Console:

  • コンソールを静音モードで開始。
  • 末尾読み上げで読む行数。
  • 抑制したイベントのログ（デバッグ）。

設定は即座に反映され保存されます。

補足

  • 静音モードは全コンソール共通です。
  • オン = イベント抑制＆読み上げキャンセル、オフ = 抑制なし。
