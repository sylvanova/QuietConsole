QuietConsole NVDA アドオン

QuietConsole はうるさいコンソールイベントを抑制し、端末出力のライブプレーンテキスト表示を提供します。

機能

  • cmd.exe、conhost.exe、powershell.exe、pwsh.exe、Windows Terminal ホスト（windowsterminal.exe、wt.exe、openconsole.exe）で動作します。
  • NVDA+Shift+C: 静音モードを切り替えます。
  • NVDA+Shift+V / NVDA+Alt+V: ライブプレーンテキスト表示を切り替えます。
  • 表示内では End で最終行先頭へ移動、F5 で末尾追従と固定行を切り替え、Esc で閉じます。
  • 設定でイベントログと強力抑制モードを有効化できます。

設定

NVDA 設定を開く: 設定 -> 環境設定 -> Quiet Console。

静音モードとプレーンテキスト表示の入力ジェスチャーは入力ジェスチャーダイアログで変更できます。

インストール

このリポジトリの 最新リリース をダウンロードし、NVDA の ツール -> アドオンの管理 -> インストール で .nvda-addon を導入してください。

トラブルシューティング

  • 再インストール後に動作が変わらない場合は NVDA を一度再起動してください。
  • ログは %LOCALAPPDATA%\Temp\nvda.log に保存されます。
