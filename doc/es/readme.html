Complemento NVDA QuietConsole

QuietConsole mantiene NVDA ágil en consolas clásicas (cmd.exe, conhost.exe) con un modo silencioso global. Activado, suprime eventos de cursor/valor/texto/cambio de nombre y cancela el habla pendiente; desactivado, la consola funciona como NVDA de serie.

Atajos

  • NVDA+Mayús+C: Alternar modo silencioso (global). Al activarlo se cancela el habla.
  • NVDA+Mayús+R: Leer las últimas 30 líneas (configurable) y llevar la revisión al final.

Configuración

NVDA → Preferencias → Configuración → Quiet Console:

  • Iniciar consolas en modo silencioso.
  • Líneas que leer con el comando de últimas líneas.
  • Registrar eventos suprimidos (depuración).

Los cambios se aplican de inmediato y se guardan.

Notas

  • El modo silencioso es global para todas las consolas.
  • Activado = se suprimen eventos y se cancela el habla; Desactivado = sin supresión.
