Extension NVDA QuietConsole

QuietConsole garde NVDA réactif dans les consoles classiques (cmd.exe, conhost.exe) grâce à un mode silencieux global. Activé, il supprime les événements caret/valeur/texte/changement de nom et annule la parole en cours ; désactivé, la console reste comme NVDA natif.

Raccourcis

  • NVDA+Maj+C : activer/désactiver le mode silencieux (global). L’activation annule la parole.
  • NVDA+Maj+R : lire les 30 dernières lignes (configurable) et placer le curseur de revue en bas.

Paramètres

NVDA → Préférences → Paramètres → Quiet Console :

  • Démarrer les consoles en mode silencieux.
  • Nombre de lignes pour la commande de lecture de fin.
  • Journaliser les événements supprimés (debug).

Les réglages s’appliquent immédiatement et sont mémorisés.

Remarques

  • Le mode silencieux est global à toutes les consoles.
  • Activé = événements supprimés et parole annulée ; Désactivé = aucune suppression.
