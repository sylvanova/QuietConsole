Extension QuietConsole pour NVDA

QuietConsole supprime les evenements bruyants de la console et fournit une vue texte brut en direct pour la sortie du terminal.

Fonctionnalites

  • Fonctionne avec cmd.exe, conhost.exe, powershell.exe, pwsh.exe et les hotes Windows Terminal (windowsterminal.exe, wt.exe, openconsole.exe).
  • NVDA+Shift+C : Basculer le mode silencieux.
  • NVDA+Shift+V / NVDA+Alt+V : Basculer la vue texte brut en direct.
  • Dans la vue : Fin va au debut de la derniere ligne, F5 bascule entre suivi du bas et ligne fixe, et Echap ferme la fenetre.
  • La journalisation optionnelle et le mode de suppression extreme sont disponibles dans les parametres.

Parametres

Ouvrez les parametres NVDA : Preferences -> Parametres -> Quiet Console.

Les gestes de saisie pour le mode silencieux et la vue texte brut peuvent etre modifies dans Gestes de commande.

Installation

Telechargez la derniere version depuis ce depot puis installez le fichier .nvda-addon via Outils -> Gerer les extensions -> Installer.

Depannage

  • Si le comportement semble ancien apres reinstallation, redemarrez NVDA.
  • Les journaux sont ecrits dans %LOCALAPPDATA%\Temp\nvda.log.
