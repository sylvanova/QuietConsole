QuietConsole NVDA 插件

QuietConsole 会抑制嘈杂的控制台事件，并提供终端输出的实时纯文本视图。

功能

  • 支持 cmd.exe、conhost.exe、powershell.exe、pwsh.exe，以及 Windows Terminal 主机（windowsterminal.exe、wt.exe、openconsole.exe）。
  • NVDA+Shift+C：切换静音模式。
  • NVDA+Shift+V / NVDA+Alt+V：切换实时纯文本视图。
  • 在视图中：End 跳到最后一行开头，F5 切换“跟随底部/固定行”，Esc 关闭窗口。
  • 可在设置中启用日志记录和极限抑制模式。

设置

打开 NVDA 设置：首选项 -> 设置 -> Quiet Console。

静音模式和纯文本视图的输入手势可在“输入手势”中修改。

安装

请从此仓库下载最新版本，然后在 NVDA 中通过 工具 -> 管理插件 -> 安装 导入 .nvda-addon 文件。

故障排除

  • 如果重新安装后行为没有变化，请重启一次 NVDA。
  • 日志写入到 %LOCALAPPDATA%\Temp\nvda.log。
