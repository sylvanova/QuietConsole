QuietConsole NVDA 插件

QuietConsole 为经典控制台（cmd.exe、conhost.exe）提供全局静音模式，让 NVDA 保持流畅。开启后抑制光标/值/文本/名称变化事件并取消正在朗读的语音；关闭后行为恢复为默认。

快捷键

  • NVDA+Shift+C：切换静音模式（全局）。开启时会取消语音。
  • NVDA+Shift+R：读取末尾 30 行（可调整）并将浏览光标移到底部。

设置

NVDA → 首选项 → 设置 → Quiet Console：

  • 启动时启用静音模式。
  • “读取末尾”命令的行数。
  • 记录被抑制的事件（调试）。

设置即时生效并会保存。

说明

  • 静音模式对所有控制台全局生效。
  • 开启 = 抑制事件并取消语音；关闭 = 不抑制。
