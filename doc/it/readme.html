Componente aggiuntivo QuietConsole per NVDA

QuietConsole sopprime gli eventi rumorosi della console e offre una vista testo semplice in tempo reale per l output del terminale.

Funzionalita

  • Funziona con cmd.exe, conhost.exe, powershell.exe, pwsh.exe e host Windows Terminal (windowsterminal.exe, wt.exe, openconsole.exe).
  • NVDA+Shift+C: Attiva/disattiva la modalita silenziosa.
  • NVDA+Shift+V / NVDA+Alt+V: Attiva/disattiva la vista testo semplice in tempo reale.
  • Nella vista: Fine va all inizio dell ultima riga, F5 alterna tra segui fondo e riga fissa, e Esc chiude la finestra.
  • Log opzionale e modalita di soppressione estrema sono disponibili nelle impostazioni.

Impostazioni

Apri le impostazioni NVDA: Preferenze -> Impostazioni -> Quiet Console.

I gesti di input per modalita silenziosa e vista testo semplice possono essere modificati in Gesti di immissione.

Installazione

Scarica l ultima versione da questo repository e installa il file .nvda-addon in NVDA tramite Strumenti -> Gestione componenti aggiuntivi -> Installa.

Risoluzione problemi

  • Se il comportamento non cambia dopo la reinstallazione, riavvia NVDA una volta.
  • I log vengono scritti in %LOCALAPPDATA%\Temp\nvda.log.
