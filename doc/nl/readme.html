QuietConsole NVDA-add-on

QuietConsole onderdrukt luidruchtige consolegebeurtenissen en biedt een live platte-tekstweergave voor terminaluitvoer.

Functies

  • Werkt in cmd.exe, conhost.exe, powershell.exe, pwsh.exe en Windows Terminal-hosts (windowsterminal.exe, wt.exe, openconsole.exe).
  • NVDA+Shift+C: Stille modus schakelen.
  • NVDA+Shift+V / NVDA+Alt+V: Live platte-tekstweergave schakelen.
  • In de weergave: End springt naar begin van de laatste regel, F5 schakelt tussen onderkant volgen en vaste regel, en Esc sluit het venster.
  • Optionele logging en extreme onderdrukking zijn beschikbaar in instellingen.

Instellingen

Open NVDA-instellingen: Voorkeuren -> Instellingen -> Quiet Console.

Invoergebaren voor stille modus en platte-tekstweergave kunnen worden aangepast in Invoergebaren.

Installatie

Download de nieuwste release uit deze repository en installeer het .nvda-addon-bestand via Extra -> Add-ons beheren -> Installeren.

Probleemoplossing

  • Als gedrag na herinstallatie niet verandert, start NVDA een keer opnieuw.
  • Logbestanden staan in %LOCALAPPDATA%\Temp\nvda.log.
