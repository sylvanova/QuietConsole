Дополнение NVDA QuietConsole

QuietConsole сохраняет отзывчивость NVDA в классических консолях (cmd.exe, conhost.exe) благодаря глобальному тихому режиму. При включении подавляются события курсора/значения/текста/смены имени и отменяется текущая речь; при выключении консоль работает как обычно.

Жесты

  • NVDA+Shift+C: переключить тихий режим (глобально). Включение отменяет речь.
  • NVDA+Shift+R: читать последние 30 строк (настраивается) и поставить обзор в конец.

Настройки

NVDA → Параметры → Настройки → Quiet Console:

  • Запускать консоли в тихом режиме.
  • Количество строк для команды чтения конца.
  • Логирование подавленных событий (отладка).

Изменения применяются сразу и сохраняются.

Примечания

  • Тихий режим общий для всех консолей.
  • Включен = события подавляются и речь отменяется; Выключен = без подавления.
