Дополнение QuietConsole для NVDA

QuietConsole подавляет шумные события консоли и предоставляет живой просмотр простого текста для вывода терминала.

Возможности

  • Работает в cmd.exe, conhost.exe, powershell.exe, pwsh.exe и хостах Windows Terminal (windowsterminal.exe, wt.exe, openconsole.exe).
  • NVDA+Shift+C: Переключить тихий режим.
  • NVDA+Shift+V / NVDA+Alt+V: Переключить живой просмотр простого текста.
  • В окне просмотра: End переходит к началу последней строки, F5 переключает режим следования за концом и фиксированной строки, Esc закрывает окно.
  • Опциональное логирование и экстремальный режим доступны в настройках.

Настройки

Откройте настройки NVDA: Параметры -> Настройки -> Quiet Console.

Жесты ввода для тихого режима и просмотра текста можно изменить в разделе Жесты ввода.

Установка

Скачайте последний релиз из этого репозитория и установите файл .nvda-addon через Инструменты -> Управление дополнениями -> Установить.

Устранение неполадок

  • Если после переустановки поведение не меняется, перезапустите NVDA.
  • Логи записываются в %LOCALAPPDATA%\Temp\nvda.log.
