QuietConsole NVDA add-on

QuietConsole suppresses noisy console events and provides a live plain-text view for terminal output.

Features

  • Works in cmd.exe, conhost.exe, powershell.exe, pwsh.exe, and Windows Terminal hosts (windowsterminal.exe, wt.exe, openconsole.exe).
  • NVDA+Shift+C: Toggle quiet mode.
  • NVDA+Shift+V / NVDA+Alt+V: Toggle live plain-text view.
  • In plain-text view: End jumps to the latest line start, F5 toggles follow-bottom/fixed-line mode, and Esc closes the view.
  • Optional logging and extreme suppression mode are available in settings.

Settings

Open NVDA settings: Preferences -> Settings -> Quiet Console.

Input gestures for quiet mode and plain-text view can be changed in Input Gestures.

Install

Download the latest release from this repository, then install the .nvda-addon file in NVDA via Tools -> Manage add-ons -> Install.

Troubleshooting

  • If behavior looks stale after reinstall, restart NVDA once.
  • Logs are written to %LOCALAPPDATA%\Temp\nvda.log.
