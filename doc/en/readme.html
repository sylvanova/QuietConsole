QuietConsole NVDA add-on

QuietConsole keeps NVDA responsive in classic consoles (cmd.exe / conhost.exe) by offering a global quiet mode. When quiet mode is on, caret/value/text/name-change events are suppressed and any pending speech is cancelled; when it is off, the console behaves like stock NVDA.

Gestures

  • NVDA+Shift+C: Toggle quiet mode (global). Turning it on cancels speech.
  • NVDA+Shift+R: Read the last 30 lines (configurable) and move the review cursor to the bottom.

Settings

NVDA → Preferences → Settings → Quiet Console:

  • Start consoles in quiet mode.
  • Lines to read with the read-last command.
  • Log suppressed events (for debugging).

All settings apply immediately and are remembered.

Notes

  • Quiet mode is global across consoles.
  • Quiet mode on = suppress events and cancel speech; off = no suppression.
